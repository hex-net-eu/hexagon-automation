== Frequently Asked Questions ==

= Do I need API keys for the AI providers? =
Yes, you'll need API keys from OpenAI (ChatGPT), Anthropic (Claude), and/or Perplexity AI. The plugin supports multiple providers so you can choose which ones to use.

= Is there a limit on content generation? =
The limits depend on your API plan with each provider. The plugin tracks usage and costs to help you monitor consumption.

= Can I schedule social media posts? =
Yes! The plugin includes a comprehensive social media scheduler with support for Facebook, Twitter, Instagram, LinkedIn, and more.

= Is the generated content SEO-optimized? =
Yes, the plugin includes automatic SEO analysis and optimization features, including keyword density, heading structure, and meta descriptions.

= Can I use this plugin on multiple sites? =
The plugin can be installed on multiple WordPress sites, but each site needs its own API keys and configuration.

= Is customer support available? =
Yes, we provide support through our website and documentation. Premium support is available for enterprise users.

== Privacy Policy ==

This plugin connects to third-party AI services (OpenAI, Anthropic, Perplexity AI) and social media platforms (Facebook, Twitter, Instagram, LinkedIn) to provide its functionality. 

**Data Processed:**
* Content prompts and generated text (sent to AI providers)
* Image generation prompts (sent to AI providers)
* Social media posts and metadata (sent to social platforms)
* Usage analytics and performance metrics (stored locally)

**Data Storage:**
* Generated content and images are stored in your WordPress database
* API usage logs and analytics are stored locally
* No personal data is sent to our servers

**Third-Party Services:**
* OpenAI API (https://openai.com/privacy/)
* Anthropic API (https://www.anthropic.com/privacy)
* Perplexity AI API (https://perplexity.ai/privacy)
* Social media platform APIs (respective privacy policies apply)

By using this plugin, you agree to the privacy policies of the connected services.
